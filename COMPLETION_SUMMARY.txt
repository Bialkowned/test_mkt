==============================================================================
                        KEY TECHNICAL DECISIONS
==============================================================================

STACK:
✅ Frontend: React 18 + Vite + Tailwind CSS
✅ Backend: Python + FastAPI
✅ Database: MongoDB 6
✅ Auth: JWT + RBAC
✅ Queue: Celery + Redis
✅ Payments: Stripe (subscriptions + Connect)
✅ Email: SendGrid
✅ Storage: AWS S3 + CloudFront CDN
✅ Deployment: Docker + Docker Compose

ARCHITECTURE DECISIONS:
✅ Stateless backend (horizontal scaling)
✅ Embedded documents in MongoDB (journeys in jobs)
✅ Event-driven background workers (Celery)
✅ Client-side screenshot annotation (Fabric.js)
✅ Hosted URL only for MVP (GitHub builds in V1)
✅ Template-based AI for MVP (OpenAI in V1)
✅ Escrow via Stripe payment intents (not wallet)
✅ 72-hour auto-release for fair payment

SECURITY:
✅ HTTPS only (TLS 1.2+)
✅ JWT with 1-hour expiry + refresh tokens
✅ Bcrypt password hashing (10 rounds)
✅ AES-256 encryption for test credentials
✅ Rate limiting (100 req/min)
✅ Input validation (Pydantic models)
✅ GDPR-compliant data handling
